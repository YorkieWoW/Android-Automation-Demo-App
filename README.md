# AutomationPracc
Demo Android Automation Testing App using Kotlin and Java.
The purpose of the app is to showcase some automation testing principles without having formal industry exposure to testing mobile apps.

Test scripts are located at:

LoginUiTest.kt // Simple login test script  
DashboardUiTest.kt // Functionality testing script

<img width="420" height="936" alt="verifying_checkboxes" src="https://github.com/user-attachments/assets/740eeda0-f98c-4c94-80a3-aa97e939d2ce" />

The project demonstrates QA Automation techniques, including waits to simulate network delay and device hitches, semantic testing and list interaction.

Languages & Tech Stack used:
- Kotlin (main app)
- Jetpack Compose
- JUnit for Unit Testing
- Espresso
- Material3
